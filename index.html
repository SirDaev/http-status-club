HTTP Status Club

Where Status is all that matters.

This service allows you to test how your code handles various HTTP response status codes. Just include the code you would like to test in the testing URL.

Example: https://httpstatus.club/code/200

This will return a status of 200 OK.

The following codes are fully supported.

200OK

The following codes are partially supported.

Most of these will return the correct response code, but probably won't return the expected headers, body, etc. Use them at your own peril!
